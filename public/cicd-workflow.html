edu-npm-packages · .github/workflows

CI/CD Workflows

4 個 GitHub Actions workflows 涵蓋完整的 CI/CD 生命週期，共用 setup-pnpm composite action 處理環境設定。

Trigger Workflow Purpose
PR → master pr.yml PR Code quality checks + conditional SAST
Push to master release.yml Release Build, publish to Azure Artifacts, deploy Storybook
Push to pre-release/** prerelease.yml Pre-release Build and publish pre-release packages
Weekly Monday 10:00 UTC+8 + manual security-scan.yml Security Full SAST scan, GitHub Issue + Teams notification

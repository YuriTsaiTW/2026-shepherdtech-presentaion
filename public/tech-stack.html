Monorepo Core
Nx
智慧型 Monorepo 建構系統。透過 Computation Cache 避免重複建構，支援 Affected 指令只跑受影響的專案。提供 task pipeline、dependency graph 視覺化，以及 Executor / Generator 擴充機制。
Build System
Code Quality
ESLint
靜態程式碼分析，強制 no-console 規則與模組邊界限制（@nx/enforce-module-boundaries）
Linter
Prettier
強制統一的程式碼格式，消除風格爭論，與 ESLint 協同運作
Formatter
Knip
偵測未使用的程式碼、exports 與 dependencies，主動清除技術債
Dead Code
Madge
視覺化模組相依圖，自動偵測循環依賴（circular dependencies）確保架構健康
Dep Graph
Git Workflow
Husky
Git hooks 自動化，在 commit / push 前自動觸發品質檢查閘道
Git Hooks
lint-staged
只對 git staged 的檔案執行 linter & formatter，大幅提升 pre-commit 速度
Pre-commit
Commitizen
互動式 Conventional Commit 問答流程，以 pnpm commit 呼叫
Commits
Conventional Changelog
解析 Conventional Commits 記錄，在發版時自動產生 CHANGELOG.md
Changelog
Documentation
Storybook
UI 元件隔離開發環境，對 design-system 的每個元件撰寫 Stories，部署至 GitHub Pages 作為互動式文件站台
UI Docs ↗
TypeDoc
從 TypeScript 原始碼的 JSDoc 註解自動產生 API 文件，與 Storybook 整合在 apps/docs 站台
API Docs
Package Registry
Azure Artifacts
Azure DevOps 托管的私有 npm registry（feed）。Nx monorepo 發布的 6 個套件（auth、design-system、hooks 等）均發布至此，各產品以 @mvb-fe scope 安裝，無需對外公開即可跨產品共用，並受 Azure AD 身份驗證保護。
Private Feed
pnpm workspace
Monorepo 套件管理器。透過 .npmrc 設定 @mvb-fe:registry=<Azure Artifacts feed URL>，本機開發與 CI pipeline 均使用相同設定，無縫安裝私有套件。
